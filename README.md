# This Application For Remover .orig file from meld

you just build and run
```go
$ go build main.go
$ ./main
Found 2 files on :
 check/jos.go.orig
 check/mantul.go.orig


(d) Deleted, (u) update list / remove from list, (q) quit
```

options:
- d: for deleted all
- u: for update list or remove path from list
- q: for quit
